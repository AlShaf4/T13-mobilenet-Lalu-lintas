Tentang Teknologi di Balik Sistem Ini

Halaman ini menjelaskan konsep dan istilah utama yang dipakai dalam pengembangan model klasifikasi rambu lalu lintas ini, mulai dari transfer learning hingga arsitektur MobileNetV2.

Apa itu Transfer Learning?

Transfer Learning adalah teknik memanfaatkan model yang sudah dilatih sebelumnya pada dataset besar (misalnya ImageNet), lalu digunakan kembali untuk tugas baru. Teknik ini menghemat waktu pelatihan dan tetap menghasilkan akurasi tinggi meski data yang tersedia terbatas.

Apa itu MobileNetV2?

MobileNetV2 adalah arsitektur convolutional neural network (CNN) buatan Google yang dirancang ringan dan cepat, menggunakan teknik depthwise separable convolutions sehingga jumlah parameter jauh lebih sedikit dibanding model seperti VGG16, tanpa mengorbankan akurasi secara signifikan. Karena ringan, model ini cocok dipakai pada sistem berbasis web maupun perangkat dengan daya komputasi rendah.

Bagaimana Model Ini Dibangun?

Base model MobileNetV2 (hasil pra-latih ImageNet) dibekukan (freeze), lalu ditambahkan layer baru: Global Average Pooling → Dense 128 (ReLU) → Dropout 0.5 → Dense 43 (Softmax). Layer tambahan inilah yang dilatih ulang menggunakan dataset GTSRB agar model bisa mengenali 43 jenis rambu lalu lintas.
